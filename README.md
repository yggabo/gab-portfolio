# Gabriel Portfolio

Un portafolio personal minimalista y profesional diseñado bajo los principios del Swiss Design (Estilo Tipográfico Internacional). Construido con el poder de Vue 3, Vite y Tailwind CSS, este proyecto prioriza la claridad, la tipografía audaz y un sistema de rejilla matemático.

---

## Principios de Diseño

Este portafolio no es solo una vitrina de proyectos, es una declaración estética:
- **Retícula de 12 Columnas:** Todo el layout sigue una estructura rigurosa que garantiza equilibrio y orden visual.
- **Tipografía Grotesk:** Uso de fuentes sans-serif claras y directas que comunican objetivamente.
- **Micro-animaciones:** Transiciones suaves que mejoran la experiencia de usuario sin distraer del contenido.
- **Dark Mode Native:** Optimizado para la legibilidad con un esquema de color de alto contraste.

---

## Stack Tecnológico

- **Framework:** [Vue 3](https://vuejs.org/) (Composition API)
- **Tooling:** [Vite](https://vitejs.org/) para un desarrollo ultrarrápido.
- **Estilos:** [Tailwind CSS v4](https://tailwindcss.com/) para un diseño atómico y eficiente.
- **Rutas:** [Vue Router](https://router.vuejs.org/) para spa navigation.
- **Iconos:** Iconografía minimalista integrada.

---

## Características Principales

- **Diseño Responsivo:** Adaptabilidad perfecta desde móviles hasta pantallas ultra-wide.
- **Grid Toggle:** Herramienta interactiva para visualizar la rejilla de diseño subyacente.
- **Smooth Scrolling:** Navegación fluida entre secciones mediante anclas.
- **Secciones Dinámicas:**
  - `Hero`: Presentación de alto impacto.
  - `About`: Perfil profesional detallado.
  - `Featured Project`: Destacado del trabajo principal.
  - `Skills`: Matriz de habilidades técnicas.
  - `Archive`: Galería completa de proyectos realizados.

## Configuración Local

Si deseas clonar y explorar este proyecto localmente:

1. **Instalar dependencias:**
   ```bash
   npm install
   ```

2. **Ejecutar en modo desarrollo:**
   ```bash
   npm run dev
   ```

3. **Compilar para producción:**
   ```bash
   npm run build
   ```
